Amazfit Bip Button Controller Privacy Policy

The Amazfit Bip Button Controller needs the following features:

  • Sound recording
Required for sound recorder function.
  • Access to photos, media and files
It is used to save the sound recorded by the sound recorder function.
  • Bluetooth function
It is used to receive button notifications from Amazfit Bip.
  • Network access

It is used for advertisement display in the app.

  • Ad ID

It is used when displaying an advertisement.
