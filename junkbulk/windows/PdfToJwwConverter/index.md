# 概要
- PDFファイルをJw_cadのjwwファイルまたはDXFファイルに変換するプログラムです。
- GUI版とCUI版があります。
- 動作環境はWindows11です。
- フリーソフトウェアです。無料で使用できますが、[開発支援](../donate/index.html)のカンパをお願いします。
