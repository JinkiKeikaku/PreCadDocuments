SakraCad file format　仕様書

SakraCadの特徴

  • SakraCadは複数ページを使用できる2DCADです。
  • 各ページにはレイヤーの他にシートがあります。シートは縮尺を設定できます。シートは一つのページに必ず一つあります。
  • シートを１ページに複数使用することで縮尺の異なる部分図を作成することができます。
  • SakraCadの図面ファイルは単純なテキスト形式で保存されています。文字コードはシフトＪＩＳです。テキストファイルなのでテキストエディタで開いて内容を確認することができます。

ファイル仕様

拡張子：skc

文字コード：shift_jis

改行コード：CRLF("\r\n")。

構造

  • １行目はヘッダー$$SakraCadText$$です。

  • ２行目以降から図面データです。図面データはタグパラメータ形式で記述されます。

    • タグパラメータはタグの次にパラメータを()で囲み記述します。パラメータは数値、文字列、タグパラメータを複数含めることができます。
      TAG1(1) ←単独の数値
      TAG2("STRING") ←単独の文字列
      TAG3(10,20)←複数の数値（座標など。複数の値はコンマで区切る。）
      TAG4(TAG1(5.5)TAG2("STRING")) ←複数のタグパラメータ

  • ヘッダーに続くトップレベルのタグパラメータはセクションです。
    FILEINFO()
    PAPER()
    LASTUSED()
    PAGES()

  • タグパラメータには省略可能なものがあります。省略された場合、デフォルト値を使用します。例えば、線幅は0がデフォルト値です。

  • ほとんどのタグパラメータの出現順は決まっていませんが、セクションは順序を守ってください。

  • 文字列を除き、空白文字（半角スペース、改行、タブ）は読み飛ばします。

  • 図形要素の座標、大きさなどのほとんどは実寸になりますが、矢印の大きさ、線幅などは用紙上の寸法（以下、用紙寸）になりますので注意してください。

  • 画像などのバイナリデータはBASE64でテキストに変換しています。なお、フラグによる指定でzlibで圧縮後にBASE64に変換することもできます。

  • 色はWindowsのGDIのCOLORREF型にもとづきます（赤：0x000000FF、緑：0x0000FF00、青：0x00FF0000）。ただし、線色及び文字色で0x01000000は透明色、0x02000000は補助色（表示のみ印刷不可）です。

  • 未知のタグパラメータは読み飛ばせるように実装してください。

  • ページ名は重複可能です。同じページ内でシート名は重複可能です。レイヤ名は重複できません（※）
    （※）現在のSakraCad（Ver0.9.4z）ではレイヤ作成時に重複をチェックして同名のレイヤは禁止しています。しかし、読み込み時はチェックしていません。

  • 個人的には図面作成時に同じページ名やシート名は付けないほうがいいと思います。

サンプル（解説付き）

$$SakraCadText$$
FILEINFO(	←FILEINFOセクション。
 ENCODING("SJIS")
 GENERATOR("SakraCad 0.9.4z")SKCVERSION("1.2.0")
)
PAPER(	←PAPERセクション
 PAPERSIZE(		←改行や字下げを入れてもOK
  NAME("A3")
  WIDTH(420)HEIGHT(297)	←HEIGHT(297)WIDTH(420)のように入れ替えてもOK
 )
 HORZ(1)
)
LASTUSED(
 PAGEINDEX   (  0 )	←パラメータやタグ前後の空白OK
)
PAGES(
 PAGE(
  NAME("page1")FLAG(3)
  LASTUSED(SHEET("sheet1")LAYER("Layer1"))
  LAYERS(
   LAYER(NAME("Layer1")FLAG(7))
  )
  SHEETS(
   SHEET(
    NAME("sheet1")
    PAPERSCALE(NAME("1:1")NUMERATOR(1))
    FLAG(768)
    LAYERS(
     LAYER(NAME("Layer1")
      SHAPES(
       LINE(
        PARAM(
         P0(2000,0)		←座標値(X,Y)はコンマで区切る
         P1(3000,1000)
        )
        ATTR(LC(0))	←線の属性。線色のみで線幅と線種は省略
      )
     )
    )
   )
  )
)

セクション解説

共通項目

数値

  • Float（実数）とInt（整数）があります。

座標・ベクトル・角度

  • 座標は数学座標でXは横軸で右が正、Yは縦軸で上が正です。
  • 座標は2つの実数をコンマで区切ります。
  • ベクトルは向きを指定するために使い書式等は座標に準拠します。
  • 角度は実数でdegreeで反時計回りが正です。

文字列

  • 文字列(String)は"（ダブルクォーテーション）で囲みます。（例 "TEXT"）
  • 文字列はraw文字列です。改行はそのまま改行です。
  • "（ダブルクォーテーション）が２つ連続した場合、単独の"（ダブルクォーテーション）として扱います。

バイナリ（Bytes）

  • BITMAP及びOLEで使用しています。
  • バイナリデータは圧縮と無圧縮があります。圧縮にはzlibを使用しています。
  • サイズには解凍後のサイズ（元サイズ）を設定します。
  • BASE64でコード化して文字列としています。72文字を超えたらコンマを付けて改行してください。
例（BITMAPより）

圧縮なし（ヘッダー）

HEADER(40,
"KAAAACAAAAAgAAAAAQAgAAAAAAAAEAAAAAAAAAAAAAAAAAAAAAAAAA=="
)

圧縮あり（32X32ドットイメージ、RGBQUAD）

IMAGE(4096,"ZLIB",
"eJztlAEKwzAIRXOa3f84u42DMagxfv3arDBWQUrSn/+MaSNyxx1HjDGmRO/RmNVA9lPmtD5q",
"3o5ZDc1ueGcaVAOzDtandZkm6IOub+EjDmIm60vfQrU3mebH+eFZOZot/OQftpr383Ek4xHx",
"K2HZuoaKR4eP2NUasnu4w+70oRJTvYBb2c+ZOq7Y9/J/JXuveFV4luvNc3z5ZMSXIDV/nfs+",
"X/doHmfhrfHuNKyxNWAvrx6mt7mvxwfn1O6t77GXz+yL5Xf6z/GvPf/u97/r/HnfPj+420gN",
"o2Pug3+JFwe3Hvw="
)

FILEINFOセクション

  • ファイル情報です

書式

FILEINFO(
	ENCODING(name)
	GENERATOR(name)
	SKCVERSION(version)
)
要素 種類 説明
ENCODING(name) name:String 文字エンコーディングの指定。SJISのみ。
GENERATOR(name) name:String ファイルを作成したアプリ名など。
SKCVERSION(version) version:String skc形式のバージョン。"1.2.0"のようにドットで区切った形式で記述。バージョン1.2.0が最新です。メジャーバージョンは大きなファイル形式の変更があり読み込めない可能性がある場合上がります。現状、メジャーバージョン1より大きいと確認ダイアログが出ます。２番目のマイナーバージョンはタグの読み飛ばしで対応できる程度の小さな変更がある場合に上がります。その場合は警告が出るのみです。

例

FILEINFO(ENCODING("SJIS")GENERATOR("SakraCad 0.9.4y")SKCVERSION("1.2.0"))

PAPERセクション[省略可]

  • 用紙情報。
  • 省略した場合は未定義（SakraCadでは読み込み前に使っていた用紙を使用）。

書式

PAPER(
 PAPERSIZE(
  NAME(name)
  WIDTH(width)
  HEIGHT(height)
 )
 HORZ(flag)
)
要素 種類 説明
NAME(name) name:String 用紙名。nameが表示される。
WIDTH(width) width:Float 用紙サイズ（幅）
HEIGHT(height) height:Float 用紙サイズ（高さ）
HORZ(flag) flag:Int 向き。1で用紙サイズの向き。0で用紙サイズの縦横入れ替えで表示。

例

PAPER(
 PAPERSIZE(NAME("A3")WIDTH(420)HEIGHT(297))
 HORZ(1)
)

LASTUSEDセクション[省略可]

  • 保存時に使っていたページ番号(最初のページが0)。
  • 省略した場合の動作は未定義（SakraCadは最後のページとなる）。

書式

LASTUSED(
 PAGEINDEX(pageNumber)
)
要素 種類 説明
PAGEINDEX(pageNumber) pageNumber:Int 最後に開いていたページ番号

例

LASTUSED(PAGEINDEX(1))

PAGESセクション

  • ページのリスト

書式

PAGES(
 PAGE(...)
 ...
)
  • PAGESはPAGE要素のリストを保持します。
  • PAGESには少なくとも１つのPAGE要素を入れてください。

PAGE要素

  • 1ページ分のデータ。

書式

PAGE(
　NAME(pageName)
　FLAG(flag)
　LASTUSED(
　　SHEET(sheetName)
　　LAYER(layerName)
　)
　LAYERS(
　　LAYER(...)
　　...
　)
　SHEETS(
　　SHEET(...)
　　...
　)
)
要素 種類 説明
NAME(pageName) pageName:String ページ名
FLAG(flag) flag:Int フラグ
1=表示可 2=印刷可
LASTUSED(
SHEET(sheetName)
LAYER(layerName)
)		 sheetName:String
layerName:String		 最後に使用していたレイヤ・シート
  • LAYERSは以下に示すレイヤ要素（レイヤ名とフラグ）のリストです。
LAYER(
 NAME(name)
 FLAG(flag)
)
要素 種類 説明
NAME(name) name:String レイヤ名
FLAG(flag) flag:Int フラグ
1=表示可 2=印刷可 4=スナップ可
  • LAYERSには少なくとも１つのLAYER要素を入れてください。
  • SHEETSはSHEET要素のリストです。
  • SHEETSには少なくとも１つのSHEET要素を入れてください。

例

PAGE(
 NAME("page1")FLAG(3)
 LASTUSED(SHEET("sheet1")LAYER("Layer1"))
  LAYERS(
   LAYER(NAME("Layer1")FLAG(7))
   LAYER(NAME("Layer2")FLAG(7))
  )
  SHEETS(
   SHEET(
    NAME("sheet1")
    FLAG(768)
    PAPERSCALE(NAME("1:100")NUMERATOR(-100))
    LAYERS(
     ...
    )
   )
   SHEET(
    NAME("sheet2")
    ...
   )      
  )
 )
)

SHEET要素

  • レイヤごとの図形を保持しています。

書式

SHEET(
 NAME(name)
 FLAG(flag)
 PAPERSCALE(NAME(name)NUMERATOR(scale))
 LAYERS(
  LAYER(...)
  LAYER(...)
  ...
 )
)
要素 種類 説明
NAME(name) name:String シート名
FLAG(flag) flag:Int フラグ
256=表示可 512=印刷
  • PAPERSCALEは縮尺です。
要素 種類 説明
NAME(name) name:String 表示などに使用する縮尺名。
NUMERATOR(scale) scale:Float 縮尺値。マイナスの場合は1/(-scale)とする。
  • LAYERS要素はレイヤのリストです。レイヤ要素を表示順で列挙します。

LAYER要素（SHEET内）

  • レイヤに含まれる図形を保持しています。

書式

LAYER(
 NAME(name)
 SHAPES(
  LINE(...)
  LINE(...)
  CIRCLE(...)
  ...
 )
)
要素 種類 説明
NAME(name) name:String レイヤ名
  • SHAPES要素は図形のリストです。図形を表示順で列挙します。

図形

共通

図形にはパラメータ(PARAM)と属性(ATTR)があります。パラメータ及び属性内要素の出現順は規定されていません。図形の書式は以下のようになります。
図形名(PARAM(...)ATTR(...))
また、一部の図形にはPARAM, ATTRの他にTEXT, SHAPES, DIB, OLECLIENTITEMのタグが現れます。例えばTEXTは文字を表示する要素にあらわれます

属性（ATTR）

属性には以下の要素があり図形毎に使用できる属性が決まっています（使用できない属性があった場合、無視されます）。

要素 種類 説明
LC(c) c:Color 線色。省略時0
LW(w) w:Float 線幅。省略時0.0 。用紙寸
LS(id) id:Int 線種。省略時0
SA(ID(id)SIZE(size)) id:Int
size:Float		 始点の矢印。省略時は矢印無し。用紙寸
EA(ID(id)SIZE(size)) id:Int
size:Float		 終点の矢印。省略時は矢印無し。用紙寸
ARROW(ID(id)SIZE(size)) id:Int
size:Float		 矢印。省略時は矢印無し。用紙寸
FC(...) 面色
TC(c) c:Color 文字色。省略時0
TA(angle) angle:Float 文字列角度。省略時0
TS(flag) flag:Int 文字列フラグ。省略時0
FONTNAME(name) name:String フォント名
FONTHEIGHT(height) height:Float フォントサイズ。実寸。
FONTWIDTHSCALE(scale) scale:Float フォント幅比率。
省略時は0（1と同じで拡縮なし）
  • Colorは32ビットの数値です。
  • 矢印サイズ、フォントサイズ等の省略時の値は未定です。
  • この他に寸法図形などはそれぞれ固有の属性を持ちます。

面色

円、ポリゴンなどの面色は単色とグラデーションがあります。

単色
要素 種類 説明
SOLID(c) c:Color 面の色
グラデーション

書式は以下のようになります。TYPEにより使う要素が違うことに注意してください（使われない要素がみつかっても無視されます）。
GRAD(TYPE(id)SC(sc)EC(ec)MID(pos,c)ANGLE(angle)P0(x,y))

要素 種類 説明
TYPE(id) id:Int 線形=0, 矩形=1, 円=2
SC(sc) sc : Color 開始色
EC(ec) ec : Color 終了色
MID(pos,c) pos:Float
c:Color		 中間色。省略時は２色のグラデーションとなる。
posは中間色の位置で0が開始色側、1が終了色側。
ANGLE(angle) angle: Float 角度。円の時は使わない。
P0(x,y) x: Float, y:Float 中心座標。線形の時は使わない。
x,yは図形領域の左上が(0, 0)、右下が(1, 1)

CIRCLE（円）

CIRCLE(
 PARAM(P0(x, y)RADIUS(radius)FLAT(flat)ANGLE(angle)) 
 ATTR(LC(...)LW(...)LS(...)FC(...))
)

パラメータ

要素 種類 説明
P0(x,y) x:Float, y:Float 中心
RADIUS(radius) radius:Float 横半径
FLAT(flat) flat:Float 扁平率（縦半径/横半径）。省略時1.0
ANGLE(angle) angle:Float 傾斜角（度）。省略時0.0

ARC（円弧）

ARC(
 PARAM(P0(x, y)RADIUS(radius)FLAT(flat)ANGLE(angle)START(start)END(end)FLAG(flag)) 
 ATTR(LC(...)LW(...)LS(...)SA(...)EA(...)FC(...))
)

パラメータ

要素 種類 説明
P0(x,y) x:Float, y:Float 中心
RADIUS(radius) radius:Float 横半径
FLAT(flat) flat:Float 扁平率（縦半径/横半径）。省略時1.0
ANGLE(angle) angle:Float 傾斜角（度）。省略時0.0
START(start) start:Float 円弧開始角（度）。省略時0.0
END(end) end:Float 円弧終了角（度）。省略時360.0
FLAG(flag) flag:Int 0で円弧、1で扇形。省略時0
  • 円弧開始角及び円弧終了角は始点から終点へ反時計回りで指定してください。また、どちらも省略可能ですが推奨しません。
  • 扇形の時、矢印は無視されます。

POLYGON（ポリゴン）

POLYGON(
 PARAM(VERTEX(V(x,y)V(x,y)...)LOOP(flag)) 
 ATTR(LC(...)LW(...)LS(...)SA(...)EA(...)FC(...))
)

パラメータ

要素 種類 説明
V(x,y) x:Float, y:Float 頂点
LOOP(flag) flag:Int 1で始点と終点を閉じます。省略時0
  • 名称はPOLYGONですが、LOOPが0の時は始点と終点が閉じずポリラインになります。
  • VERTEXは頂点のリストです。
  • LOOPが１の場合は属性の矢印は無視されます。

SPLINE（スプライン）

SPLINE(
 PARAM(VERTEX(V(x,y)V(x,y)...)LOOP(flag))

)

パラメータ

要素 種類 説明
V(x,y) x:Float, y:Float 頂点
LOOP(flag) flag:Int 1で始点と終点を閉じます。0は閉じません。省略時0
  • VERTEXは頂点のリストです。
  • LOOPが１の場合は属性の矢印は無視されます。

TEXT（１行文字）

TEXT(
 PARAM(P0(x,y)) 
 ATTR(LC(...)LW(...)LS(...)FC(...)FONTNAME(...)FONTHEIGHT(...)FONTWIDTHSCALE(...)TC(...)TA(...)TS(...))
 TEXT(SRC(text))
)

パラメータ

要素 種類 説明
P0(x,y) x:Float, y:Float 配置点
  • 文字の書かれる位置はTSにより変わるので注意。

文字

要素 種類 説明
TEXT(SRC(text)) text:String 文字列
  • 文字列中の改行は無視されます。

MULTITEXT（複数行文字）

MULTITEXT(
 PARAM(P0(x, y)WIDTH(width)HEIGHT(height)) 
 ATTR(LC(...)LW(...)LS(...)FC(...)FONTNAME(...)FONTHEIGHT(...)FONTWIDTHSCALE(...)TC(...)TA(...)TS(...))
 TEXT(SRC(text))
)

パラメータ

要素 種類 説明
P0(x,y) x:Float, y:Float 配置点。左下
WIDTH(width) width:Float 幅。省略時20
HEIGHT(height) height:Float 高さ。省略時20
  • 文字の書かれる位置はTSの揃えのフラグにより変わるので注意。配置点は左下固定。
  • 幅及び高さの省略は推奨されない。

文字

要素 種類 説明
TEXT(SRC(text)) text:String 文字列
  • 文字列中の改行が有効となります。
  • FONTHEIGHTは実寸です。

LABEL（引き出し線）

LABEL(
 PARAM(VERTEX(V(x,y)V(x,y)...)TEXTPOINT(x,y)FLAG(flag))

  LC(...)LW(...)LS(...)FC(...)TC(...)TS(...)
  LABELSTYLE(...)
 )
TEXT(SRC(text)) 
)

パラメータ

要素 種類 説明
V(x,y) x:Float, y:Float 頂点
TEXTPOINT(x,y) x:Float, y:Float 文字の配置点（文字列の中央）。座標は縮尺を割った値（用紙上の座標）であることに注意。 FLAGが1の時は省略出来ます。
FLAG(flag) flag:Int フラグ １の場合、文字自動配置（文字は水平配置で最後の頂点から文字下に線を引く）。0の場合、下線なしで文字位置角度等自由。省略時は1
  • VERTEXは頂点のリストです。2点以上必要。
  • 文字の配置基準は文字列中央で用紙寸です。
  • TEXTPOINTが用紙座標系なのは実装ミスです。影響が大きそうなので修正はしません。
  • 普通は文字自動配置を使うと思います。自分には自由配置の使い道はありませんでした（作っておいて申し訳ないのですが）。文字の自由配置は後からつけたのでTEXTPOINTの座標系がおかしなことになりました。
  • 自動配置のみ使いTEXTPOINTは省略したほうがいいと思います。
